# OSCAR Patch Mining

**From Fix Commits to Vulnerable Functions: Automated Patch Mining for Function-Level CVE Resolution**

This project extends the patch mining technique developed within the OSCAR platform into a standalone research contribution, targeting **MSR 2027** (Mining Software Repositories) or an equivalent venue.

## Status

- **Phase**: Pre-research (planning stage)
- **Priority**: Post-ICSME submission (after May 28, 2026)
- **Target Venue**: MSR 2027 Data Showcase or Technical Track

## Contents

- `research-plan.md` — Detailed research plan, timeline, and methodology
- `src/` — Mining scripts (to be developed)
- `data/` — Mined datasets (to be populated)
- `evaluation/` — Validation results and analysis
